# Store Management

Manage your merchant stores.

| Action | Method | Endpoint |
| :--- | :--- | :--- |
| **List Stores** | `GET` | `/stores/` |
| **Create Store** | `POST` | `/stores/` |
| **Retrieve Store** | `GET` | `/stores/{id}/` |
| **Update Store** | `PATCH` | `/stores/{id}/` |
| **Delete Store** | `DELETE` | `/stores/{id}/` |
